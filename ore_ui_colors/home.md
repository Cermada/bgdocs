### What is this page about?

In Apr 16, 2025, a Youtube channel `@8Crafter` released a video called [Ore UI Customizer](https://www.youtube.com/watch?v=i4JSs1_a_tQ), showcasing his [website](https://www.8crafter.com/utilities/ore-ui-customizer) that proving Ore UI is still customizable even if it's hardcoded.

when i stumbled upon his website. there are a lot of things that made me confused - especially the color configuration.

<img src="/images/oreuicustomizer/oreuicustomizer.png" alt="preview">

This documentation specifically made for Ore UI Customizer with an intent to giving notes for every color which one actually appears in the game vs what apppears on the website. *this project is made to company you throughout ore ui customization.*

this doc would become part of `Bedrock GUI Docs` with the release of this beta build, i'm excited yet exhausted to see things i would do for future updates.
